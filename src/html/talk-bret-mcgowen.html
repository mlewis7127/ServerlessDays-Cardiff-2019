Bret McGowen

Bret is a Developer Advocate for Serverless on the Google Cloud Platform team at Google, focusing on serverless products like Google Cloud Functions, App Engine, Firebase, machine learning APIs, and more. He's currently an aspiring Node.js developer

Talk: Serverless beyond functions: Google's mobile, big data, and machine learning serverless tools

Bret will cover Google Cloud's serverless compute, big data and machine learning tools, as well as Firebase, Google's serverless backend for mobile and web developers. Firebase can help you add powerful features to your applications, with everything from real time data updates, user authentication, monetizing your apps, and more. Learn how to leverage the power of Google to quickly build autoscaling apps - all without managing servers or application runtimes.
